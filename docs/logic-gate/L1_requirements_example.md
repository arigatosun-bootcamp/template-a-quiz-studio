# L1 要件分解（作成例）

## 1. 目的（誰の・何の困りごとを解決する？）

- **誰が困っている？**: 日本に住んでいるが、自分の国の地理・観光名所・食文化をよく知らない人
- **何に困っている？**: 教科書やネット検索で「覚える」だけの勉強はつまらなくて続かない
- **どう解決する？**: AIが毎回違うクイズを自動生成し、ゲーム感覚で日本の知識を身につけられるWebアプリを作る
- **「できたらOK」と言える状態**: ユーザーがログインし、好きなジャンル・難易度を選んでクイズ5問に回答でき、結果と過去の履歴が確認でき、日本地図上で自分の進捗が視覚的にわかること

## 2. やること / やらないこと

### 今回やること
- ユーザー認証（メール＆パスワードでの登録・ログイン・パスワードリセット）
- 3つのジャンルから選択できる（地理 / 観光名所 / ご当地グルメ）
- 3段階の難易度から選択できる（初級 / 中級 / 上級）
- AIが選んだジャンル・難易度に合わせてクイズを5問自動生成する
- 3択の選択肢から回答する
- 回答後に正解・不正解と解説が表示される
- 5問終了後にスコア（5問中何問正解）を表示する
- 過去の回答履歴を一覧で確認できる
- トップ画面にSVGの日本地図を表示し、正解した都道府県に色がつく
- 難易度によって地図の色が変わる（初級=薄い / 中級=普通 / 上級=濃い）
- 1日の利用回数に上限を設ける（20回/日）
- アカウント削除機能

### 今回やらないこと（次回以降 or 対象外）
- 問題を指定して出題する機能 → 今回はAI自動生成のみに集中するため
- 日本についての検索機能 → クイズアプリであり検索サイトではないため
- SNSシェア機能 → ユーザー数が増えてからフェーズ2で検討する
- マイページ（プロフィール編集） → 最低限の設定画面のみ作る
- ランキング・リーダーボード → 複数ユーザー間の比較は今回のスコープ外
- 多言語対応 → まず日本語のみで完成させる

## 3. 使い方の流れ（ユーザー目線で）

- **流れ1（初回利用）**: ユーザーがアプリのURLにアクセスする → ログイン画面が表示される → 「新規登録」を押してメールとパスワードを入力する → 確認メールが届く → メール内のリンクをクリックする → ログイン画面に戻りメールとパスワードでログインする → トップ画面が表示される
- **流れ2（クイズをプレイする）**: トップ画面でジャンル（地理/観光/グルメ）を選ぶ → 難易度（初級/中級/上級）を選ぶ → 「ゲームスタート」を押す → AIが5問生成する（ローディング表示あり） → 1問ずつ3択が表示される → 選択肢を押すと正解/不正解と解説が出る → 「次の問題」で進む → 5問終了後に「5問中○問正解」が表示される → 「トップに戻る」でトップ画面に戻る
- **流れ3（履歴を見る）**: トップ画面で「回答履歴」を押す → 過去のゲーム一覧が新しい順に表示される → 1件を押すとそのゲームの5問の問題・自分の回答・正解・解説が見られる → 「戻る」で一覧に戻れる
- **流れ4（達成状況を見る）**: トップ画面で「達成状況」を押す → 日本地図が表示され、正解した都道府県に色がついている → 統計情報（総プレイ数、正解率など）が表示される

## 4. 保存が必要なデータ

- **ユーザー情報**（メールアドレス）: ログイン認証に必要。次回ログイン時にデータを紐づけるため
- **クイズセッション**（ジャンル、難易度、スコア、プレイ日時）: どのゲームでどの難易度を選び何点だったかを記録するため。履歴表示に使う
- **クイズ問題**（問題文、選択肢3つ、正解、解説、対象都道府県）: AIが生成した問題を保存して履歴で見返せるようにするため。同じ問題を復習できる
- **ユーザーの回答**（選んだ選択肢、正解かどうか、回答日時）: どの問題にどう答えたかを記録するため。正解率の計算に使う
- **都道府県の達成状況**（都道府県名、クリアした最高難易度）: 日本地図に色をつけるため。どの都道府県をどの難易度で正解したかを記録する

## 5. 完了条件（「これができたらOK」を具体的に）

- 条件1（認証）:
  - メールとパスワード（8文字以上）で新規登録ができる
  - 確認メールが届き、リンクをクリックすると認証が完了する
  - ログイン後にトップ画面が表示される
  - どうやって確認する？：実際にメールアドレスで登録し、確認メールのリンクを踏み、ログインしてトップ画面が出ることを目視で確認する

- 条件2（クイズプレイ）:
  - ジャンルと難易度を選んでスタートすると5問のクイズが表示される
  - 問題生成中はローディングが表示される（10秒以内に生成完了する）
  - 各問題に3つの選択肢がある
  - 回答するとすぐに正解/不正解と解説が表示される
  - 5問完了後に「5問中○問正解」のスコアが表示される
  - どうやって確認する？：3ジャンル × 3難易度 = 9パターンでプレイし、すべてスムーズに動くことを確認する

- 条件3（履歴・達成状況）:
  - 回答履歴画面に過去のゲームが新しい順に表示される
  - 1件タップすると5問の詳細（問題・自分の回答・正解・解説）が見られる
  - 達成状況画面で日本地図に色がついた都道府県が表示される
  - どうやって確認する？：複数回プレイした後に履歴画面と達成状況画面を開き、データが正しく反映されていることを確認する

- 条件4（利用制限）:
  - 1日20回プレイすると「本日の上限に達しました」と表示される
  - 翌日0:00にリセットされて再度プレイできる
  - どうやって確認する？：利用回数のAPIレスポンスで remaining が 0 になることを確認する
